Responsive Stylus micro framework

Mint Settings

Let's start with the global settings in Mint, look like this.

  • mint.col_count property indicates the number of columns in the grid, you can supply any number.
  • mint.container_max_width property shows the maximum width that is the default in the container function.
  • mint.break object contains 4 breakpoints for media queries.
  • mint.offset object contains internal indentations for each of the stop points.

Stylus

// _settings.styl
mint = {
	col_count: 12,
	container_max_width: 1300px,
	break: {
		lg: 1300px,
		md: 960px,
		sm: 780px,
		xs: 560px,
	},
	offset: {
		all: 30px,
		md: 20px,
		sm: 2%,
		xs: 1%,
	},
}

Mint API

Container function

container([ int max_width = mint.container_max_width ]) - Sets the default properties for the container

Parameters

  • max_width - Maximum width of the container (px)

Examples:

.container
	container(960px)

Yields:

.container {
  max-width: 960px;
  padding-left: 30px;
  padding-right: 30px;
  margin: 0 auto;
}
@media screen and (max-width: 1300px) {
  .container {
    padding-left: 30px;
    padding-right: 30px;
  }
}
@media screen and (max-width: 960px) {
  .container {
    padding-left: 20px;
    padding-right: 20px;
  }
}
@media screen and (max-width: 780px) {
  .container {
    padding-left: 2%;
    padding-right: 2%;
  }
}
@media screen and (max-width: 560px) {
  .container {
    padding-left: 1%;
    padding-right: 1%;
  }
}

Flex-row function

flex-row() - Reduction of properties display: flex; and flex-wrap: wrap;

Examples:

.container
	flex-row()

Yields:

.container {
  display: flex;
  flex-wrap: wrap;
}

Col function

col( int count [, string break = 'all'] ) - Defines a column

Parameters

  • count - Column width
  • break - The breakpoint of resolution. Possible values: lg, md, sm, xs

Examples:

Example #1

.col
  col( 2 )
  col( 2, 'lg' )
  col( 3, 'md' )
  col( 6, 'sm' )
  col( 12, 'sm' )

Yields:

.col {
  width: calc( 16.666666666666668% - 30px );
  padding-left: 15px;
  padding-right: 15px;
}
@media screen and (max-width: 1300px) {
  .col {
    width: calc( 16.666666666666668% - 30px );
    padding-left: 15px;
    padding-right: 15px;
  }
}
@media screen and (max-width: 960px) {
  .col {
    width: calc( 25% - 20px );
    padding-left: 10px;
    padding-right: 10px;
  }
}
@media screen and (max-width: 780px) {
  .col {
    width: calc( 50% - 2% );
    padding-left: 1%;
    padding-right: 1%;
  }
}
@media screen and (max-width: 780px) {
  .col {
    width: calc( 100% - 2% );
    padding-left: 1%;
    padding-right: 1%;
  }
}

Example #2 - without second parameter

.col
  col( 2 )

Yields:

.col {
  width: calc( 16.666666666666668% - 30px );
  padding-left: 15px;
  padding-right: 15px;
}

Example #3 - with second parameter

.col
  col( 2, 'lg' )

Yields:

@media screen and (max-width: 1300px) {
  .col {
    width: calc( 16.666666666666668% - 30px );
    padding-left: 15px;
    padding-right: 15px;
  }
}

Example #3.1 - alternative syntax

.col
  col 2 'lg'

Yields:

@media screen and (max-width: 1300px) {

    width: calc( 16.666666666666668% - 30px );

  }
}

Breakpoint function

lg( string property, string value ) - Adds this selector with the specified property in the media query of the corresponding breakpint

Similar functions

  • md()
  • sm()
  • xs()

Parameters

  • property - Propery name
  • value - Property value

Examples:

Example #1

.col
  lg(display, none)
  md(display, block)
  sm(display, flex)
  xs(display, inline-block)

Yields:

@media screen and (max-width: 1300px) {
  .col {
    display: none;
  }
}
@media screen and (max-width: 960px) {
  .col {
    display: block;
  }
}
@media screen and (max-width: 780px) {
  .col {
    display: flex;
  }
}
@media screen and (max-width: 560px) {
  .col {
    display: inline-block;
